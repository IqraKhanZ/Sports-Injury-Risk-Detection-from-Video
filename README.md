# Sports Injury Risk Detection from Video

## Objective
The objective of this project is to develop an automated system that detects and analyzes injury risks in sports videos. By leveraging computer vision and deep learning techniques, the application aims to identify high-risk movements, improper biomechanics, and potential injury events to help athletes and coaches train more safely.

## Tech Stack
- **Backend**: FastAPI (Python), Uvicorn, Pydantic, python-dotenv
- **Frontend**: React.js (Vite, Tailwind CSS)
- **AI/ML & Computer Vision**: Python libraries for video processing and pose estimation (to be integrated)
- **Database**: SQLite / PostgreSQL (to be integrated)

## Project Structure
- `backend/`: FastAPI application handling API requests, model inference, and database interactions.
- `frontend/`: React single-page application for uploading videos, visualizing analysis results, and tracking metrics.
